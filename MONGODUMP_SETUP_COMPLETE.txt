✅ CÁC TÍNH NĂNG:

Backup:
  ✓ Tạo backup ngay với timestamp
  ✓ Tính toán kích thước tự động
  ✓ Tạo metadata.json cho mỗi backup
  ✓ Log chi tiết hoạt động

Restore:
  ✓ Chọn backup từ danh sách
  ✓ Hỏi xác nhận trước restore
  ✓ Mode MERGE (thêm/update) hoặc DROP (xóa cũ)
  ✓ Khôi phục đầy đủ dữ liệu

Quản Lý:
  ✓ Liệt kê tất cả backup
  ✓ Xem chi tiết từng backup
  ✓ Xóa backup cũ
  ✓ Theo dõi dung lượng

Tự Động:
  ✓ Backup theo lịch (daily, weekly, monthly)
  ✓ Auto-cleanup backup cũ (giữ 30 bản)
  ✓ Windows Task Scheduler hoặc Node-Cron
